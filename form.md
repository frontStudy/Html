# Form 
## form 기본 양식
```
<form action="">
    <p> -: <input type="text" name="id"></p>
    <p> -- : <input type="password" name="pwd"></p>
    <input type="submit">
</form>
```
* form action에는 'submit'을 통해 제출한 정보가 보내지는 주소가 들어감
* input type이 'text'면 입력하는 정보가 화면상에 나타나고, 'password'면 모자이크 처리
